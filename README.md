## Run Locally

Clone the project

```bash
  git clone https://github.com/JDCodeWork/emoji-game.git
```

Go to the project directory

```bash
  cd emoji-game
```

Install dependencies

```bash
  npm install
```

Start the development server

```bash
  npm run dev
```
